# Quest 2 VR 手部术后康复游戏设计草案

## 目标与约束
- 目标：术后康复，训练**精细动作**（拇指-食指捏合、指尖独立控制、稳定性）。
- 平台：Meta Quest 2。
- 输入：手柄（Touch Controllers）。
- 需要记录训练数据（过程指标 + 结果指标）。
- 体验偏向游戏化与趣味性。

## 交互与技术栈建议
- 交互：基于 **XR Interaction Toolkit (XRI)** 的 Grab/Socket/Direct Interactor。
- HurricaneVR：可选（若已有成熟抓取/手部动画系统也可集成），但本项目仅需手柄精细动作，可优先使用 XRI，减少依赖复杂度。
- 数据记录：本地保存（JSON/CSV），可后续扩展到云端。

## 关卡设计（4-5 关）

### 1) 微型拼图工坊（精细抓取 + 放置）
**目标**：拾取小型拼图块并放入匹配凹槽。
- 训练点：稳定捏合、精准对位、手部稳定。
- 玩法：3D 形状匹配 + 轻提示（高亮凹槽）。
- 评分：对位误差、完成时间、掉落次数。

### 2) 旋钮修理台（指尖旋转控制）
**目标**：转动多个小旋钮到目标角度。
- 训练点：小幅度旋转控制、持续动作。
- 玩法：每个旋钮有目标角度，转到位即点亮。
- 评分：角度误差、平均稳定度、耗时。

### 3) 按键序列（精准点击）
**目标**：按下小按钮完成序列。
- 训练点：指尖精准定位、节奏控制。
- 玩法：类似 Simon Says，按提示顺序完成。
- 评分：错误次数、反应时间、平均偏差。

### 4) 微型搬运车（双手协作）
**目标**：双手抓住微型托盘，稳定搬运小球到终点。
- 训练点：双手协调、稳定控制。
- 玩法：托盘必须保持水平，小球不可掉落。
- 评分：水平稳定度、掉落次数、完成时间。

### 5) 组合挑战（综合训练）
**目标**：从抓取、旋转、放置到按键的小流程。
- 训练点：综合协调与流程记忆。
- 玩法：迷你任务链，完成后给予评分。
- 评分：各项指标综合加权。

## 数据记录设计
每次训练输出一条记录（本地 JSON/CSV）。

### 记录字段（建议）
- `session_id`
- `level_id`
- `start_time` / `end_time`
- `duration_seconds`
- `attempts`
- `errors`
- `drop_count`
- `avg_precision`（对位/角度误差）
- `avg_stability`（抖动/偏差）
- `score`

## 任务系统与评分
- 每关 0–100 分。
- 使用**时间、精度、稳定度、错误次数**综合评分。
- 训练记录可用于康复进度评估。

## Quest 2 性能与交互注意事项
- 目标 72 FPS。
- 控制物体数量与物理复杂度。
- 视觉提示简单清晰（对比度高、引导明确）。
- 避免快速镜头移动，减少眩晕。
